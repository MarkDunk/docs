# Project Structure

```text
/
    app/  ===> this is where all of the app javascript files are
    assets/ ===> this is where all of the apps assets are located (ie images)
    build/ ====> this is the output directory of running `npm run build`
    config/ ===> all of the default configuration settings
    logs/ ===> output of the log files
    node_modules/ ===> output directory of running `npm install`
    styles/ ===> CSS directory
    views/ ===> HTML views directory
    index.html ===> main index file. Starting point of application
    package.json ===> Describes the application, including the dependencies
```
